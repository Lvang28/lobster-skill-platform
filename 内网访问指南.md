# 龙虾 Skill 平台 - 内网访问完整指南

## 🎯 问题诊断

您提到"阿里内网的其他人也无法访问"，这通常是因为：

1. **公司防火墙限制** - 禁止非标准端口（5000）
2. **VLAN 隔离** - 不同部门在不同网段
3. **macOS 防火墙** - 系统自带防火墙阻止入站连接
4. **网络 ACL** - 访问控制列表限制

## ✅ 解决方案（按推荐程度排序）

### 方案一：Tailscale 虚拟局域网 ⭐⭐⭐⭐⭐

**最推荐** - 完全不受物理网络限制，加密传输，稳定可靠。

#### 优点：
- ✅ 无需开放任何防火墙端口
- ✅ 穿透 NAT 和公司网络限制
- ✅ WireGuard 加密，安全可靠
- ✅ 免费使用（20 台设备 +3 用户）
- ✅ 配置简单，5 分钟完成

#### 部署步骤：

**1. 服务器端（您的机器）**
```bash
# 安装
brew install tailscale

# 启动
sudo tailscale up

# 复制输出的 URL 到浏览器登录

# 查看分配的 IP
tailscale ip
# 输出类似：100.87.45.123
```

**2. 同事端（每位访问者）**
```bash
# macOS
brew install tailscale
sudo tailscale up

# Windows: 下载安装包 https://tailscale.com/download
# Linux: sudo apt install tailscale
```

登录**同一个 Tailscale 账号**（或加入同一网络）。

**3. 访问平台**
```
http://[您的 Tailscale IP]:5000
例如：http://100.87.45.123:5000
```

详细配置见 `README-TAILSCALE.md`

### 方案二：检查并关闭 macOS 防火墙 ⭐⭐⭐

如果同事和您在**同一办公网**（相同 VLAN），可能是 macOS 防火墙阻止了访问。

#### 临时关闭防火墙测试：
```bash
sudo /usr/libexec/ApplicationFirewall/socketfilterfw --setglobalstate off
```

#### 或者添加例外规则：
```bash
# 系统偏好设置 → 安全性与隐私 → 防火墙 → 防火墙选项
# 添加 Python 或允许传入连接
```

#### 测试连通性：
让同事执行：
```bash
curl -I http://[您的 IP]:5000
```

如果超时，说明防火墙确实阻止了。

### 方案五：FRP 内网穿透（企业级）⭐⭐

如果您有阿里云 ECS 服务器：

**1. 在阿里云 ECS 上部署 frps：**
```bash
# 下载 frp
wget https://github.com/fatedier/frp/releases/download/v0.50.0/frp_0.50.0_linux_amd64.tar.gz
tar -xzf frp_*.tar.gz
cd frp_*

# 配置 frps.ini
cat > frps.ini <<EOF
[common]
bind_port = 7000
EOF

# 启动
./frps -c frps.ini
```

**2. 本地配置 frpc：**
```bash
# 下载 macOS 版 frp
wget https://github.com/fatedier/frp/releases/download/v0.50.0/frp_0.50.0_darwin_amd64.tar.gz

# 配置 frpc.ini
cat > frpc.ini <<EOF
[common]
server_addr = [您的阿里云公网 IP]
server_port = 7000

[lobster]
type = tcp
local_ip = 127.0.0.1
local_port = 5000
remote_port = 8080
EOF

# 启动
./frpc -c frpc.ini
```

**3. 同事访问：**
```
http://[阿里云公网 IP]:8080
```

## ❓ 常见问题

**Q: Tailscale 安全吗？**
A: 非常安全。使用 WireGuard 协议端到端加密，只有加入您网络的人才能访问。

**Q: 会影响公司网络安全策略吗？**
A: 不会。Tailscale 是出站连接，不违反任何入站防火墙规则。

**Q: 为什么直连不稳定？**
A: 公司网络可能有 QoS 限制、VLAN 隔离或动态 IP 分配。Tailscale 通过虚拟局域网绕过这些问题。

**Q: 可以几个人同时访问吗？**
A: 可以。Flask 支持并发连接，Tailscale 免费版支持 3 个用户同时在线。
