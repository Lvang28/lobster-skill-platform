## 🚀 三种外网访问方法

### 方法一：局域网访问（最简单，推荐！）

**适用：** 同一 WiFi/办公室的朋友

**步骤：**

1. **查看本机 IP**
   ```bash
   # macOS
   ipconfig getifaddr en0
   
   # 输出示例：192.168.1.100
   ```

2. **启动平台**
   ```bash
   cd lobster-skill-platform
   ./start.sh
   ```

3. **告诉朋友访问地址**
   ```
   http://你的IP:5000
   例如：http://192.168.1.100:5000
   ```

4. **朋友在手机/电脑浏览器打开即可！**

---

### 方法二：Cloudflare Tunnel（推荐远程访问）

**适用：** 任何地方的朋友（跨城市/国家）

**一键启动：**
```bash
cd lobster-skill-platform
./start-public.sh
```

**会自动：**
1. ✅ 安装 cloudflared（如果未安装）
2. ✅ 启动平台
3. ✅ 生成公网 URL
4. ✅ 显示可分享的链接

**示例输出：**
```
🌐 公网访问地址：
   https://abc123def456.trycloudflare.com

📱 把这个 URL 发给朋友即可访问！
```

**优点：**
- ✅ 免费
- ✅ 无需注册
- ✅ 一键启动
- ✅ 全球可访问

---

### 方法三：手动安装 ngrok

**如果一定要用 ngrok：**

1. **手动下载安装：**
   - 访问：https://ngrok.com/download
   - 下载对应版本
   - 解压到 `/usr/local/bin/ngrok`

2. **启动：**
   ```bash
   # 终端 1
   ./start.sh
   
   # 终端 2
   ngrok http 5000
   ```

3. **复制生成的 URL 分享给朋友**
