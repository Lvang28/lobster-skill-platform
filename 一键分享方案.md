## 🚀 方案对比

### 方案 A：ngrok（最快，5 分钟搞定）⭐⭐⭐⭐⭐

**优点：**
- ✅ 立即生成公网 URL
- ✅ 无需注册（临时使用）
- ✅ 任何人直接访问
- ✅ 适合演示和测试

**缺点：**
- ⚠️ 免费版每次重启 URL 会变
- ⚠️ 有访问次数限制

**操作步骤：**

```bash
# 1. 安装 ngrok
brew install ngrok

# 2. 先启动龙虾平台（如果还没启动）
cd /Users/lvang/.qoderwork/workspace/mnfoi60u4zvyxkz5/lobster-skill-platform
source venv/bin/activate
python app.py &

# 3. 在新终端运行 ngrok
ngrok http 5000
```

会输出类似：
```
Forwarding: https://abc123def456.ngrok.io -> http://localhost:5000
```

**把这个 URL 发给任何人即可！**

---

### 方案 B：Cloudflare Tunnel（稳定，免费）⭐⭐⭐⭐

**优点：**
- ✅ 永久固定 URL
- ✅ 完全免费
- ✅ 不限流量
- ✅ Cloudflare 大品牌可靠

**缺点：**
- ⚠️ 需要注册 Cloudflare 账号（免费）

**操作步骤：**

```bash
# 1. 安装 cloudflared
brew install cloudflared

# 2. 运行隧道（首次需要登录）
cloudflared tunnel --url http://localhost:5000
```

第一次运行会：
1. 打开浏览器让您登录 Cloudflare（免费）
2. 生成一个固定的 `.trycloudflare.com` URL
3. 以后每次用同一个 URL

**把 URL 发给任何人即可！**

---

### 方案 C：Render 云部署（最专业）⭐⭐⭐⭐⭐

**优点：**
- ✅ 永久固定 URL
- ✅ 24 小时在线
- ✅ 无需您开电脑
- ✅ 完全免费
- ✅ 专业云服务

**缺点：**
- ⚠️ 需要 GitHub 账号
- ⚠️ 首次部署需要 10 分钟

**操作步骤：**

```bash
cd /Users/lvang/.qoderwork/workspace/mnfoi60u4zvyxkz5/lobster-skill-platform

# 1. 初始化 Git
git init
git add .
git commit -m "Lobster platform"

# 2. 上传到 GitHub
# 在 github.com 创建新仓库，然后：
git remote add origin https://github.com/YOUR_USERNAME/lobster-skill.git
git branch -M main
git push -u origin main

# 3. 访问 render.com
# - 用 GitHub 账号登录
# - 点击 New+ → Blueprint
# - 选择刚才的仓库
# - 自动部署完成！
```

获得永久 URL：`https://lobster-skill-xxxx.onrender.com`

## ❓ 常见问题

**Q: ngrok 的 URL 会变吗？**  
A: 免费版每次重启会变。如果想固定，可以注册 ngrok 账号（免费）。

**Q: 能多少人同时访问？**  
A: ngrok 免费版支持 40 个并发连接，完全够用。

**Q: 我关电脑后别人还能访问吗？**  
A: 不能。如果需要 24 小时在线，使用 Render 云部署。

**Q: 哪个方案最省心？**  
A: Render - 部署一次，永久在线，不用管它。
