# 🔄 完全重新部署 Render

## 🎯 步骤（重要！）

### Step 1: 删除当前服务

1. 访问 https://dashboard.render.com
2. 点击 `lobster-skill-platform`
3. 点击左侧 **"Settings"**
4. **滚动到页面最底部**
5. 找到红色区域 **"Danger Zone"**
6. 点击 **"Delete Web Service"**
7. 输入服务名称确认删除
8. 点击确认

---

### Step 2: 创建新的 Web Service

1. 返回 Dashboard 首页
2. 点击 **"New +"** → **"Web Service"**
3. 选择 **"Connect a repository"**
4. 找到并选择 `lobster-skill-platform` 仓库
5. 点击 **"Connect"**

---

### Step 3: 配置新服务

**基本信息：**
```
Name: lobster-skill-platform
Region: Singapore
Branch: main
Root Directory: (留空)
Runtime: Python 3
Build Command: pip install -r requirements.txt
Start Command: gunicorn app:app --log-file -
Instance Type: Free
```

**环境变量（关键！）：**

点击 **"Advanced"** → **"Add Environment Variable"**：

```
Key: SECRET_KEY
Value: lobster_render_2024_unique_key_abc789xyz
```

```
Key: FLASK_DEBUG
Value: false
```

**⚠️ 注意：** SECRET_KEY 换一个不同的值！

---

### Step 4: 创建并等待

1. 点击 **"Create Web Service"**
2. 等待 5-8 分钟
3. 看到绿色 "Live" 标志
4. 访问 URL 测试

---

- 使用全新的环境
