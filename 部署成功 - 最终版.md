## 💡 优化建议

### 避免冷启动（推荐）

使用免费的 Uptime Robot 保持服务器活跃：

1. 访问 https://uptimerobot.com
2. 注册免费账号
3. 添加监控：
   ```
   URL: https://lobster-skill-platform-v2.onrender.com
   Interval: 5 minutes
   ```
4. 这样服务器就不会休眠了！

### 添加数据库持久化

为了防止数据丢失（SQLite 在 Render 重启后会重置）：

1. Render Dashboard → **New +** → **PostgreSQL**
2. 选择 **Free** 套餐
3. 创建数据库
4. 复制 **Internal Database URL**
5. 在 Web Service → **Environment** 添加：
   ```
   DATABASE_URL = [粘贴的 URL]
   ```
6. 手动重新部署

## ❓ 常见问题

**Q: 访问显示 500 错误？**  
A: 可能是冷启动或临时问题，等待 1 分钟后刷新

**Q: 数据会丢失吗？**  
A: SQLite 会在服务重启后重置。建议添加 PostgreSQL

**Q: 可以自定义域名吗？**  
A: 可以！在 Render Settings → Custom Domains 添加

**Q: 有人数限制吗？**  
A: 免费版支持足够的并发访问（日常使用完全够用）

**Q: 访问速度慢？**
