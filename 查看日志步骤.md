# 📋 如何查看 Render 完整日志

## 🎯 步骤详解

### Step 1: 进入服务页面

访问：https://dashboard.render.com  
找到并点击 `lobster-skill-platform`

---

### Step 2: 切换到 Logs 标签

在左侧菜单中：
1. 点击 **"Logs"**（在 Overview 下面）
2. 确保顶部下拉选择的是 **"All logs"**（不是 "Build logs"）

---

### Step 3: 查看完整错误

**向下滚动页面**，找到红色的错误信息。

应该类似这样：
```
[Error] Traceback (most recent call last):
  File "...", line XX, in <module>
    ...
[Error] SomeError: Error message here
```

---

### Step 4: 截图或复制错误

**请截图包含以下内容：**
1. 完整的错误堆栈（从 "Traceback" 开始）
2. 最后一行的错误类型和信息
3. 如果有很长，可以分多次截图

**或者：**
1. 点击日志区域右上角的 **"Download"** 按钮（如果有）
2. 下载完整日志文件
3. 用文本编辑器打开
4. 搜索 "Error" 或 "Traceback"
5. 复制相关部分发给我

---

## 🔍 常见错误位置

日志中找这些关键词：
- `Traceback` - Python 错误堆栈开始
- `Error` - 错误类型
- `Exception` - 异常
- `Failed` - 失败信息

---

## 💡 如果找不到 Logs

**可能的位置：**
1. 左侧菜单 → **Logs** → 选择 **All logs**
2. 或者 Overview 页面下方也有简略日志

---

## 📤 发送给我

截图或复制错误信息后，告诉我：
1. 错误类型（如 `ImportError`, `KeyError` 等）
2. 错误信息（最后一行）
3. 出错的文件名和行号（如果有）

我会帮您修复！🦞
