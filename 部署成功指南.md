### 添加数据库（推荐）
为了让数据持久化：

1. Render Dashboard → **New +** → **PostgreSQL**
2. 选择 **Free** 套餐
3. 创建数据库
4. 复制 **Internal Database URL**
5. 在 Web Service → **Environment** 添加环境变量：
   ```
   DATABASE_URL = [粘贴刚才复制的 URL]
   ```
6. 手动触发重新部署
