# 🔍 检查 Render 环境变量

## 📋 重要步骤

### Step 1: 进入 Environment 页面

1. 访问 https://dashboard.render.com
2. 点击 `lobster-skill-platform` 服务
3. 点击左侧菜单的 **"Environment"**

---

### Step 2: 检查环境变量

**必须有以下变量：**

| Key | Value |
|-----|-------|
| `SECRET_KEY` | `lobster_platform_2024_secret_key_xyz123` |
| `FLASK_DEBUG` | `false` |

**可选但推荐：**
- 如果添加了 PostgreSQL 数据库，还需要 `DATABASE_URL`

---

### Step 3: 如果没有或不确定

**添加/修改变量：**

1. 点击 **"Add Environment Variable"**
2. 添加第一个：
   ```
   Key: SECRET_KEY
   Value: lobster_platform_2024_secret_key_xyz123
   ```

3. 再点击 **"Add Environment Variable"**
4. 添加第二个：
   ```
   Key: FLASK_DEBUG
   Value: false
   ```

5. 点击 **"Save Changes"**

---

### Step 4: 重新部署

保存后：
1. 返回 **"Overview"** 页面
2. 点击右上角 **"Manual Deploy"** 按钮
3. 等待 5-8 分钟

---

## ❓ 常见问题

**Q: 已经有这些变量了怎么办？**  
A: 即使有，也请删除后重新添加一次（确保值正确）

**Q: 还有其他变量吗？**  
A: 不需要其他变量了，这两个是必须的

**Q: 重新部署后还是失败？**  
A: 请查看 Logs → All logs，找到完整错误信息发给我
